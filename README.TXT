To execute program, run the code and enter timesteps, maximum wattage for the building, and the name of the output text file. Afterwards, a menu option will appear with options to add, delete, list appliances, and either start the simulation or quit the program. To add an appliance, enter a into the console and input information regarding the appliance (instructions will follow on the console). To delete an appliance, enter d into the console and afterwards enter the uniqueID of the appliance you wish to delete (it is recommended to list all appliances first in order to find the unique ID easily). To list all the appliances onto the console, enter l. To start the simulation, enter s. Finally, to end the program, enter q.
